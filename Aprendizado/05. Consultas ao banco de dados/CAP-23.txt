Neste capítulo o professor falou sobre o Query Methods.


- Query Method é aquele "macete" que o nome do método presente na interface Repository já faz a consulta, por exemplo: findByEmail(String email). O 
professor recomenda utilizar esse tipo apenas em consultas simples;


- Para saber quais Query Methods existem você deve procurar a documentação so Spring: https://docs.spring.io/spring-data/jpa/docs/current/reference/html/#jpa.query-methods
